# Password Revealer

Features
* Show / Hide Password;
* Verify password strength:
    * 1-3 characters = BAD;
    * 4-6 characters = MEDIUM
    * Above 7 characters = GOOD

Technologies used:
* HTML
* CSS
* JavaScript Functions
